## Forecasting Dashboard

This is a simple **Next.js** project built with **Tailwind CSS**, **Recharts**, and **TypeScript**. It demonstrates a single-page dashboard layout with chart and interactive UI components.

### Getting Started

#### 1. Clone the repo

```
git clone https://github.com/shelyaa/brimmo_agency.git
cd brimmo_agency
```

#### 2. Install dependencies

```
npm install
```

#### 3. Run the project locally

```
npm run dev
```
